# FormMate AI

**FormMate AI** is a smart assistant designed to help users automatically extract text from scanned documents and fill out forms — especially useful for scholarship applications and other academic documents.

## Features

- Upload scanned PDFs or images (JPEG, PNG)  
- Extract text using OCR (Optical Character Recognition)  
- Preview extracted text easily  
- Designed to simplify the form-filling process  

## How to Use

1. Upload your scanned document on the app’s main page.  
2. The AI extracts and displays the text.  
3. Use the extracted information to fill your forms faster.  

## Tech Stack

- Streamlit for the web app interface  
- Python libraries: pytesseract, pdfplumber, Pillow  
- Hosted on Streamlit Community Cloud
